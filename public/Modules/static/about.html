CSS3 + Angular + Routing

How cool are CSS3 transitions? Very cool.
How cool is AngularJS? Moderately...moderately cool.

This app uses 3d transformations to show different pages.
Each 'page' is a different face on the cube.

The cube is built as an Angular directive so that you can easily use it in your project.
